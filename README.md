## stats-card-ui

### How to run local

```bash
# Clone this repository
$ git clone https://github.com/korayguler/stats-card-ui

# Go into the repository
$ cd stats-card-ui


# Install dependencies
$ npm install

# Start the parcel
$ npm start

# For build
$ npm run build

```

### Tech Stack

- parcel
- sass

### Prerequisite

- node
- npm
