## stats-card-ui

### How to run localside

```bash
# Clone this repository
$ git clone https://github.com/korayguler/stats-card-ui

# Go into the repository
$ cd pinterest-clone


# Install dependencies
$ npm install

# Start the parcel
$ npm start

# For build
$ npm run build

```

### Tech Stack

- parcel
- sass

### Prerequisite

- node
- npm
